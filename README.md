# cv-typescript

Simple cv-typescript example

## Overview
This repository showcases a simple example of a CV built using TypeScript.

## Features
- Responsive design
- Easy customization

## Technologies
- TypeScript
- Node.js

## Usage
To run the CV example:
1. Install dependencies: `npm install`
2. Run the application: `npm start`
